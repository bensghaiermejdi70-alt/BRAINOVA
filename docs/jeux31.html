Règles de NeuroSphere

🎯 Objectif

NeuroSphere est un jeu de stratégie futuriste où vous affrontez une IA avancée. Le but est de contrôler le maximum de territoire en plaçant stratégiquement vos pièces sur la grille 3D.

🎮 Comment jouer

  • Cliquez sur une case vide pour placer votre pièce (cyan)
  • L'IA placera automatiquement sa pièce (magenta)
  • Formez des lignes, colonnes ou diagonales pour marquer des points
  • Les cases spéciales (dorées) donnent des points bonus
  • Atteignez l'objectif de points pour passer au niveau suivant

🤖 Niveaux d'IA

  • Facile: IA basique, idéale pour débuter
  • Moyen: IA tactique, analyse les patterns
  • Difficile: IA stratégique, anticipe vos mouvements

📊 Système de score

  • Placement normal: +10 points
  • Ligne complète: +50 points
  • Case spéciale: +25 points
  • Combo multiple: points x2

⌨️ Contrôles

  • Clic gauche: Placer une pièce
  • Espace: Pause/Reprendre
  • Échap: Fermer les menus
  • R: Redémarrer la partie
