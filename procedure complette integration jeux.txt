Procédure standard pour intégrer un nouveau jeu X (méthode “jeux4”)
1. Base de travail

On part toujours du dernier fichier global corrigé (par ex. index_global_fullscreen_game11_fixed.html pour ajouter le jeu 12).

On a le fichier jeuxX.html (code complet du jeu à intégrer).

2. Modification de la carte

Repérer la carte correspondante dans la plateforme (exemple : carte 12 → <div class="game-number">12</div>).

Modifier uniquement l’attribut onclick du <div class="game-card"> parent pour appeler une nouvelle fonction :

<div class="game-card" onclick="launchGameX()">
   <div class="game-number">X</div>
   ...
</div>


⚠️ Ne pas toucher aux autres cartes.

3. Création de l’overlay plein écran

Juste avant </body>, ajouter un bloc overlay contenant :

un <iframe srcdoc="..."> avec le code complet du fichier jeuxX.html (tout HTML, CSS, JS intégré directement),

un bouton rouge ✖ pour fermer et revenir à la plateforme.

Exemple inséré :

<!-- Overlay for Game X -->
<div id="gameXOverlay" style="display:none; position:fixed; top:0; left:0; width:100%; height:100%; background:#000; z-index:10000;">
  <button onclick="closeGameX()" 
          style="position:absolute; top:20px; right:20px; background:red; color:#fff; border:none; padding:10px; font-size:20px; cursor:pointer; z-index:10001;">✖</button>
  <iframe id="gameXFrame" style="width:100%; height:100%; border:none;" srcdoc="...contenu jeuxX.html échappé..."></iframe>
</div>

4. Fonctions JavaScript

Ajouter dans un <script> :

function launchGameX() {
  document.getElementById('gameXOverlay').style.display = 'block';
}

function closeGameX() {
  document.getElementById('gameXOverlay').style.display = 'none';
  // reset l’iframe pour relancer le jeu proprement
  var ifr = document.getElementById('gameXFrame');
  if (ifr) {
    try { ifr.srcdoc = ifr.srcdoc; } catch(e) { ifr.src = 'about:blank'; }
  }
}

// Permet de fermer aussi avec la touche Échap
document.addEventListener('keydown', function(e) {
  if (e.key === "Escape") {
    closeGameX();
  }
});

5. Vérifications

Cliquer sur la carte X → le jeu s’ouvre en plein écran.

Le bouton rouge ✖ → ferme l’overlay et ramène à la plateforme.

La touche Échap → ferme aussi l’overlay proprement.

Les autres cartes (1 → X-1) restent inchangées.

✅ Résultat : un fichier global autonome qui contient tous les jeux (1 → X).
